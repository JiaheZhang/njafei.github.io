ES6-Symbols

发表于 2018-02-22 | 分类于 ES6

#背景
ES5中，属性都可以通过点方法来获取和赋值。这也带来了一个问题，当我想给一个类增加属性的时候，除非我看完这个类的所有细节，否则我很难知道一个属性名是否已经被占用。而ES6为了解决这个问题，引入了一个新的原始类型Symbol，它表示独一无二的值，来避免重复的命名。

加入Symbol之后，JS中的基础类型共有：

  • undefined
  • Null
  • Boolean
  • String
  • Number
  • Object
  • Symbol
