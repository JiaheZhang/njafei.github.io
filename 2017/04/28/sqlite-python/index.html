操作相关

数据库的操作和git还是有点相似的，都是有提交，回滚等，我们依次介绍下：

  • connection.commit() 提交修改，否则不会保存
  • connection.rollback() 回滚上次的commit
  • connection.close() 关闭数据库连接
  • connection.total_changes() 查看修改的总行数
